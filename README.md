# AI Interview Frontend

A modern React-based frontend for the AI Interview Platform. This application provides an intuitive user interface for conducting and managing AI-powered interviews.

## Features

- 🎯 Interactive interview dashboard
- 🔐 Secure user authentication
- 📊 Interview analytics and results
- 💬 Real-time interview session management
- 🎨 Responsive and user-friendly UI
- 🚀 Built with Next.js for optimal performance

## Project Structure

```
src/
├── app/
│   ├── dashboard/          # Dashboard pages
│   ├── login/              # Authentication pages
│   ├── layout.tsx          # Root layout
│   ├── page.tsx            # Home page
│   ├── globals.css         # Global styles
│   └── favicon.ico         # Favicon
```

## Prerequisites

- Node.js 18+ or Bun
- npm/yarn/bun package manager

## Installation

1. Clone the repository:
```bash
git clone https://github.com/swapnendukishoreduttadutta-maker/ai-interview-frontend.git
cd ai-interview-frontend
```

2. Install dependencies:
```bash
npm install
# or
bun install
```

## Development

Start the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Building for Production

```bash
npm run build
npm run start
# or
bun run build
bun run start
```

## Technologies Used

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** CSS Modules
- **Package Manager:** npm/yarn/bun

## API Integration

This frontend connects to the AI Interview Backend API. Ensure the backend is running before starting the frontend.
